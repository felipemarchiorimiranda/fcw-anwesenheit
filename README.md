# Anwesenheitsverwaltung FC Wiesendangen

**by Felipe Marchiori**

Dashboard zur Verwaltung der Anwesenheiten der 2. Mannschaft des FC Wiesendangen.

## Live-URL

Nach dem Einrichten von GitHub Pages ist das Tool erreichbar unter:
```
https://DEIN-USERNAME.github.io/fcw-anwesenheit/
```

## Features

- Anwesenheitsverwaltung Training & Spiele
- Langzeitplanung mit Übersicht
- Spielerstatistik & Spielerdetail-Export
- Abwesenheitsgründe erfassen
- Serientermine erstellen
- Google Sheets Sync (alle Staff sehen dieselben Daten)
- Saison-Verwaltung
